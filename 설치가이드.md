# 인별위젯 설치 가이드

개발 몰라도 됩니다. 순서대로만 따라하세요!

---

## STEP 1. Supabase 가입 & DB 만들기

1. https://supabase.com 접속 → 무료 가입
2. "New Project" 클릭 → 프로젝트 이름 입력 (예: sns-clip) → 비밀번호 설정 → Create
3. 프로젝트 생성 후 좌측 메뉴 "SQL Editor" 클릭
4. 아래 SQL 전체 복사해서 붙여넣고 "Run" 클릭:

```sql
create table stores (
  id uuid default gen_random_uuid() primary key,
  mall_id text unique not null,
  access_token text,
  refresh_token text,
  expires_at text,
  script_tag_no integer,
  created_at timestamptz default now()
);

create table videos (
  id uuid default gen_random_uuid() primary key,
  mall_id text not null,
  src_url text not null,
  thumb_url text,
  handle text,
  caption text,
  insta_url text,
  sort_order integer default 0,
  is_active boolean default true,
  created_at timestamptz default now()
);
```

5. 좌측 메뉴 "Project Settings" → "API"
   - Project URL 복사해두기 → NEXT_PUBLIC_SUPABASE_URL
   - anon public 키 복사 → NEXT_PUBLIC_SUPABASE_ANON_KEY
   - service_role 키 복사 → SUPABASE_SERVICE_ROLE_KEY

---

## STEP 2. GitHub에 코드 올리기

1. https://github.com 가입
2. "New repository" → 이름: sns-clip-app → Create
3. https://github.com/apps/desktop 에서 GitHub Desktop 설치
4. GitHub Desktop에서 "Add Local Repository" → sns-clip-app 폴더 선택
5. Publish repository (비공개로 해도 됨)

---

## STEP 3. Vercel에 배포하기

1. https://vercel.com 가입 (GitHub 계정으로 로그인)
2. "New Project" → GitHub에서 sns-clip-app 선택 → Import
3. "Environment Variables" 섹션에 아래 값들 입력:

| 이름 | 값 |
|------|-----|
| CAFE24_CLIENT_ID | 개발자센터 Client ID |
| CAFE24_CLIENT_SECRET | 개발자센터 Client Secret Key (보기 클릭해서 복사) |
| NEXT_PUBLIC_SUPABASE_URL | Supabase Project URL |
| NEXT_PUBLIC_SUPABASE_ANON_KEY | Supabase anon key |
| SUPABASE_SERVICE_ROLE_KEY | Supabase service_role key |
| NEXT_PUBLIC_APP_URL | (아직 모름, 일단 비워두고 배포 후 채우기) |
| CAFE24_REDIRECT_URI | (아직 모름, 일단 비워두고 배포 후 채우기) |

4. Deploy 클릭 → 완료되면 주소 확인 (예: https://sns-clip-app.vercel.app)

---

## STEP 4. 나머지 환경변수 채우기

배포 완료 후 Vercel 주소를 알게 됩니다.

Vercel → 프로젝트 → Settings → Environment Variables에서:
- `NEXT_PUBLIC_APP_URL` = `https://sns-clip-app.vercel.app` (실제 주소)
- `CAFE24_REDIRECT_URI` = `https://sns-clip-app.vercel.app/auth/callback`

저장 후 Vercel에서 "Redeploy" 클릭.

---

## STEP 5. 카페24 개발자센터 설정

1. https://developers.cafe24.com 로그인
2. 내 앱 → 해당 앱 선택
3. **App URL**: `https://sns-clip-app.vercel.app` (실제 주소)
4. **Redirect URI**: `https://sns-clip-app.vercel.app/auth/callback`
5. **권한 추가**: 
   - "권한선택" 에서 스크립트태그(Scripttag) → 읽기+쓰기 추가
6. 저장

---

## STEP 6. 테스트 실행

카페24 개발자센터 → "테스트 실행" 클릭 → 정상 작동 확인 → 심사요청

---

## 완료 후 사용법

앱 설치하면 관리자 페이지가 자동으로 열립니다.

1. 카페24 파일 매니저에 MP4 영상 업로드
2. 관리자 페이지에서 URL 붙여넣기
3. 인플루언서 계정명, 캡션 입력
4. 쇼핑몰 상품 페이지에 위젯 자동 표시!
